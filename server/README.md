Dr Websight Server
===================

A server application to ping given URL and return the status in JSON format.

Requirements
-------------

- Node.js. Tested with 4.x series.

Installation
-------------

- install node.js
- npm -v
- if your npm is still 2.x: npm -g update npm
- npm install

Running
--------

- npm run start

Server now accepts traffic to http://localhost:1235
